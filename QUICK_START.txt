Issue: Database connection failed
Fix:  • Verify SQL Server LocalDB is running:
        sqllocaldb start MSSQLLocalDB
      • Check database name: Testsanitization

Issue: API authentication failed
Fix:  • Verify .env file exists
      • Check GITHUB_COPILOT_TOKEN in .env
      • Test: python -c 'import os; from dotenv import load_dotenv; load_dotenv(); print(bool(os.getenv(\"GITHUB_COPILOT_TOKEN\")))'

Issue: No PII columns detected
Fix:  • Database might have no PII data
      • Review schema manually
      • Add columns manually to config

Issue: Pydantic LogConfig error
Fix:  • Use simplified configs (production.json)
      • Or run AI detection directly (doesn't need full config)
